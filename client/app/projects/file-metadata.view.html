File Metadata Microservice

This microservice displays file metadata at different stages of an upload.

Additional Info



See It In Action

Upload a file to test metadata inspection. (Hosted externally for security)




Additional Information

The query string is mandatory, the offset and count are optional. The default count is 10 with an offset of 0.

Use Limitations

This application makes use of Herokus free tier. Accordingly, the application is limited to:

  • 18 hours / day of run time

Input Limitations

This service currently accepts only one file at a time.

User Story

The following user stories were the basis for this application:

  1. I can submit a FormData object that includes a file upload.
  2. When I submit something, I will receive the file size in bytes within the JSON response
